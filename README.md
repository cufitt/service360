# CUFitt Service360

**Execution with precision.**

CUFitt Service360 is an open-source, credit-union-specific data model and accelerator for **Microsoft Dynamics 365 Customer Service**, designed to deliver a true 360-degree member view for frontline service teams.

## Why Service360?
Credit unions using Dynamics 365 often struggle with generic CRM entities that do not reflect core credit union concepts such as shares, dividends, householding, and loan relationships. Service360 fills this gap with a purpose-built, community-driven extension.
